## 📖 Sobre el Proyecto
Este repositorio alberga el verdadero motor de datos y comunicaciones detrás de **App Flow Chat**. Consiste en un servidor robusto escrito en Node.js que gestiona las conexiones en tiempo real de millones de mensajes que viajan de un dispositivo a otro, respaldado de forma segura y veloz usando **Socket.io**.

Junto a los endpoints de conexión y chat, el servicio sirve estáticamente una **página web de aterrizaje ultra pulida y minimalista**. Esta página está construida con lo mejor de HTML semántico, CSS (Flexbox/Grid), y JS Vanilla, proveyendo al usuario una excelente presentación visual sin la sobrecarga de frameworks frontend extensos.

---

## ⚡ Características Principales

- ⚡️ **Mensajería Instantánea**: Uso intensivo de WebSockets (Socket.io) para eliminar las esperas, latencia o recargas simuladas en los chats.
- 🎨 **Minimalismo Elevado**: Interfaz web recreada bajo los estándares UI modernos usando estéticas súper limpias ("clean & airy") y enfocadas a la experiencia nativa.
- 📱 **Diseño 100% Responsivo**: Flujo adaptable en pantallas móviles, tablets y monitores gigantescos.
- 🔗 **Integración Precisa**: Trabaja en absoluta armonía reaccionando a los eventos de envío y recepción producidos por el cliente Flutter de `App-Flow-Chat`.

## 🔧 Guía de Instalación

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/Fabian-Lugo/Backend-Flow-Chat.git
   ```
2. **Entrar a la carpeta y descargar librerías**:
   ```bash
   cd Backend-Flow-Chat
   npm install
   ```
3. **Poner el servidor en órbita**:
   ```bash
   npm run start:dev
   ```

Una vez encendido tu servidor local, visita `http://localhost:3000` (o el puerto configurado si corresponde) en tu navegador para interactuar con la moderna *Landing Page*, o conecta la aplicación Flutter para probar el envío de mensajes.
